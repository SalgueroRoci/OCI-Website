Identity Access Management

Lets you control who has access to your cloud resources. You can control what type of access a group of users have and to which specific resources. IAM components (compartments, users, groups, policies, and federation) help administer and keep track of your OCI resources easier.

Governance

Part of Identity and Access Management. Oracle Cloud Infrastructure offers governing tools to help administer and keep track of Oracle Cloud resources. From tags, cost tags, adn audit logs help keep track of Oracle resources in your organization.

Networking

On Oracle Cloud provides you with Virtual Cloud Networks (VCN) for your resources as well as other components to help with different connectivity models. These tutorials will help you with the basics of networking from VCN , subnetting, route tables, security list, and internet gateway.

Compute

Oracle lets you provision and manage compute hosts, known as instances. Can use instances to host your applications and access it securely from your computer. In these tutorials will walkthrough the different instances Oracle Cloud offers.

Block Storage

Block Volume service lets you provision more storage you can attach to your compute instances. You can create, attach, connect, and move volumes as needed for your applications and storage requirments. You can use the colume like a regular hard drive.

Object Storage

Object Sotrage is a storage platform that can store an unlimited amount of unstructed data of any content type. Can be used to safely and securely store or retrieve data directly from the internet of cloud platforms.

File Storage

Oracle provides a network file system that can access among instances. OCI instances and On-Prem machines can access File Storage system using fast connect and VPN. Supports NFSv3 protocol and Network Lock Manager (NLM) protocol for file locking.

Databases

Oracle provides Oracle Databases on Oracle Cloud where you can deploy familiar Oracle Databases on the cloud which are reliable and secure.

Advanced Networking

Advanced Networking includes tutorials outside of basic VCN configuration and resources. Basic understanding of VCNs, route tables, and security lists are recommended. Certain topics on Private/Public Load Balancer, NAT Gateway, Service Gateway, and VCN Local and Remote Peering will be covered.
